jQuery 1.1.3 Opacity Bug

Animation with opacity changes fail in IE6 and IE7 with the new jQuery 1.1.3 release. The opacity style is not properly altered, just the display style at the end of the animation. This also affects .fadeIn() and .fadeOut(), which call the .animate() function.
