Description

var aHtml='<div id="x"><iframe src="http://localhost..../"/></div>'

var dlg = $(aHtml).dialog(...close:function(){dlg.remove()})

the invocation of:

dlg.remove()

causes jquery to make a request to the iframe-src.

dlg.empty().remove() helped us out.
