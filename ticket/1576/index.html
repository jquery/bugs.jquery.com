Description

When I use the Slider in safari, something odd happens. When I drag it, it moves 2 pixels upwards.

Tempoary fix:

	onSlide: function(cordx, cordy, x, y) {
	  if ($.browser.safari) {
	    $('.slider').css('margin-top', '2px');
	  }
	},
	onChange: function(cordx, cordy, x, y) {
	  if ($.browser.safari) {
	    $('.slider').css('top', '0px');
	  }
	}
