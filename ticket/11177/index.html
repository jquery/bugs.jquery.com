Description

I've noticed an issue with the latest version of jQuery in both Chrome and Firefox that could be troublesome. Firefox simply points out the issue when you do certain things like inspecting the code in a page, but Chrome locks up. I suspect the issue itself is something either related to a plugin that is not compatible with this latest version of jQuery, or the possibility that the browsers themselves are subject to choke points brought on by other incompatibilities.

In any event the issue only seems to occur in sites using using WP 3.3+, and only when an attempt is made to use browser development tools to inspect code.

A couple of examples where I have seen this issue occur are:

[url]http://friendshipcircleinternational.com/bike/[/url]
