Changed January 28, 2014 10:07PM UTC by Rodrigo Rosenfeld Rosas <rr.rosas@gmail.com> comment:11

Oops, I posted it too soon. The return value was wrong in the previous examples. Fixed version of workaround:

(function(){

if (window.getDefaultComputedStyle) {
  var old = getDefaultComputedStyle;
  window.getDefaultComputedStyle = function(elem) {
    if (var style = old(elem)) return style.display;
    return {display: $.css(elem, "display")};
  };
}

})();
