Description

problem with jQuery adding <form> tags to DOM.

eg:

$('#obj').html("<form ...><input ...></form>");

in this example, all the elements with in the form tags will be added to the DOM, but the form tags themselves will be skipped.

expects:

<div id='obj'>

<form ...><input ...></form>

</div>

results:

<div id='obj'>

<input ...>

</div>
