Description

the html tags is as:

<select>

<option value="good">good</option>

<option value="good1">good1</option>

<option value="good2">good2</option>

<option value="good3">good3</option>

<option value="good4">good4</option>

</select>

my codes is as:

var obj = $("select");

var temp = obj.clone(true);

$("body").append(temp);

$("option").each(function(){
